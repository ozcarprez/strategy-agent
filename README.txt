STEP 1: Install Streamlit and OpenAI
-------------------------------------
1. Open Command Prompt (Windows) or Terminal (Mac).
2. Run this:
   pip install streamlit openai

STEP 2: Set Your OpenAI API Key
-------------------------------
You need an OpenAI account (https://platform.openai.com/signup)

1. After creating your account, go to:
   https://platform.openai.com/account/api-keys

2. Copy your API key (starts with 'sk-...')

3. In the terminal, run this (replace with your key):
   On Mac/Linux:
     export OPENAI_API_KEY=sk-...

   On Windows:
     set OPENAI_API_KEY=sk-...

STEP 3: Run the App
-------------------
In the folder where this file is, run:

   streamlit run streamlit_strategy_agent.py

Your browser will open the app automatically!

Enjoy your Strategy Agent!
